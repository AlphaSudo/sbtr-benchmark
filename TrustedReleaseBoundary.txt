

# Final Verdict + Complete Proof + Comparison

## The Decision: TRB Wins, But Absorb Rule 4 From SBTR

TRB is the stronger foundation because "Artifact Quarantine" is a harder, more specific rule than generic "output validation." But SBTR's output validation catches a real injection attack that TRB doesn't explicitly address.

**The final framework: TRB + Output Validation = 6 rules.**

```
Rule 0: PIN          — immutable SHA references
Rule 1: QUARANTINE   — untrusted lane gets no secrets, no write
Rule 2: ISOLATE      — trusted lane is separate, protected, first-party only
Rule 3: REBUILD      — trusted lane rebuilds from source fresh
Rule 4: QUARANTINE ARTIFACTS — only metadata crosses the boundary, never binaries
Rule 5: VALIDATE     — any output crossing the boundary is sanitized
```

Public name: **Secretless Build, Trusted Release**
Technical name: **Trusted Release Boundary**

That's final. Now let's prove it.
